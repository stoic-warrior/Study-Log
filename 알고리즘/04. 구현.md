# 아이디어를 코드로 바꾸는 구현
- **구현 :** 머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정
- "풀이는 떠오르는데 코드로 옮기기 어렵다"라고 느끼는 문제들이 이 유형
## 피지컬로 승부하기
- 책에서는 구현 능력을 **피지컬**에 비유하고 있습니다.
  - **개념:** 머릿속에 있는 알고리즘을 소스코드로 정확하고 빠르게 바꾸는 과정입니다.
  - **비유:** 스타크래프트에서 전략(알고리즘)을 잘 짜도 마우스 컨트롤(구현)이 안 되면 지는 것과 같습니다.
  - **특징:** 이론적으로는 쉽지만, 막상 코드로 옮기려면 조건이 까다롭거나 코드가 길어져서 실수하기 쉬운 유형입니다.
- 초보자가 특히 어려워하는 '구현' 유형은 보통 다음과 같은 특징을 가집니다.
  - **지나치게 긴 코드:** 알고리즘은 단순한데 처리해야 할 조건문이 너무 많을 때.
  - **까다로운 출력 포맷:** 특정 소수점 자리까지 출력하거나, 특수 문자를 포함해야 할 때.
  - **문자열 파싱:** 입력받은 문자열을 특정 단위로 끊어서 리스트에 넣는 등의 작업.
  - **방대한 라이브러리 지식:** 예를 들어 '순열'을 구할 때 직접 구현할 수도 있지만, 파이썬의 itertools 같은 표준 라이브러리를 모르면 시간이 훨씬 오래 걸립니다.
- 이 책에서는, 2가지를 구현 유형으로 다루고 있다
  - **완전 탐색 (Brute Force) :** 모든 경우의 수를 주저 없이 다 계산하는 방식
  - **시뮬레이션 (Simulation) :** 문제에서 제시한 알고리즘을 한 단계씩 차례대로 수행
- 구현 문제는 코딩 테스트 환경 제약을 알아야 하므로 아래서 다룬다 
## 구현 시 고려해야 할 메모리 제약 사항
### 변수의 표현 범위
- Java에서 int는 약 21억까지 담을 수 있다 (4바이트)
- 값이 커진다 싶으면 그냥 long을 쓰자 (8바이트)
### 배열의 크기
- 문제에서 보통 128MB같은 조건이 붙는다
- int 배열 크기 1,000만이 약 40MB 정도이다. 
## 채점 환경
- 보통
  - 시간 제한 : 1초
  - 메모리 제한 : 128MB
## 구현 문제에 접근하는 방법
- 문제의 길이가 길고 사소한 입력 조건이 많습니다.
- 고차원적인 사고력보다는 문법에 익숙함이 중요하며, 지레 겁먹지 않고 문제 요구사항을 그대로 코드로 옮기는 능력이 필요합니다.
- 문자열 처리, 큰 정수 처리 등이 자주 출제됩니다. (따라서 파이썬이 유리)

```
// N+1 정사각형 배열 만들기
// 방향벡터 만들기
// for문 돌려서 입력된 문자열이 몇번째 인덱스인지 파악
// 그 인덱스의 방향벡터 방향으로 이동
// 경계선넘으면 continue
public class Main{

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
      
        int N = Integer.parseInt(br.readLine());
        int c = 1;
        int r = 1;

        StringTokenizer st = new StringTokenizer(br.readLine());
        int[] dc = {-1,1,0,0};
        int[] dr = {0,0,-1,1};
        char[] ch = {'L','R','U','D'};
        while(st.hasMoreTokens()) { // 발상
            char nextMove = st.nextToken().charAt(0);

            for(int i=0;i<4;i++) {
                if(ch[i]==nextMove) {
                    int nc = c + dc[i];
                    int nr = r + dr[i];

                    if(nc<1 || nc>N || nr<1 || nr>N) break;

                    c = nc;
                    r = nr;
                } 
            }
        }

        bw.write(r+" "+c);
        bw.flush();
        bw.close();


    }
}
```

- 시간복잡도 : $O(M)$
- M : 여기서 $M$은 이동 횟수, 즉 StringTokenizer로 읽어오는 명령어의 개수

```
/*
1. 왼쪽으로 회전 후 2번으로
2. 전진할 수 있으면 전진(visit+1, 회전스택 리셋), 없으면 회전스택1쌓고, 회전스택 4개면 3번으로, 아니면 1번으로
3. 회전스택 리셋하고, 뒤로 후진, 뒤로못가면 종료
*/
public class Main{

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken()); // 세로크기 3~50
        int M = Integer.parseInt(st.nextToken()); // 가로크기

        st = new StringTokenizer(br.readLine());
        int currRow = Integer.parseInt(st.nextToken()); // 0부터 시작
        int currCol = Integer.parseInt(st.nextToken()); // 0부터 시작
        int dir = Integer.parseInt(st.nextToken()); // 0123 북동남서

        int[][] map = new int[N][M]; // 0육지1바다
        boolean[][] visit = new boolean[N][M]; // 방문여부
        visit[currRow][currCol] = true; // 현재위치 방문
        int visitCnt = 1;  // 현재위치 방문
        for(int i=0;i<N;i++) {
            st = new StringTokenizer(br.readLine());
            for(int j=0;j<M;j++) {
                map[i][j] = Integer.parseInt(st.nextToken());
            }
        }

        int stack = 0; 
        int[] dr = {-1,0,1,0};
        int[] dc = {0,1,0,-1};
        boolean exit = false;
        while(true) {
            // 1
            dir = (dir + 3) % 4; // 왼쪽으로 회전
            
            //2
            int nR = currRow + dr[dir];
            int nC = currCol + dc[dir];

            if(map[nR][nC]==0 && !visit[nR][nC]) {
                visit[nR][nC] = true;
                currRow = nR;
                currCol = nC;
                visitCnt++;
                stack = 0; // 이걸 처음에 놓쳤다. 전진하면 회전스택도 초기화해야한다
            }
            else{
                stack++;
            }

            // 3
            if(stack==4) {
                stack = 0;


                nR = currRow - dr[dir];
                nC = currCol - dc[dir];

                if(map[nR][nC]==0) { // 육지면 후진
                    currRow = nR;
                    currCol = nC;
                
                }
                else{ // 바다면 종료
                    exit = true;
                }
                    
            }

            if(exit) break;


        }

        bw.write(visitCnt+" ");
        bw.flush();
        bw.close();


    }
}
```

- 오답노트 : 전진하면 회전스택 초기화하는걸 놓쳤다
- 상하좌우 문제의 경우 char로 방향을 정해야해서 for문을 돌렸지만, 이 문제는 그냥 dir을 인덱스로 박으면 된다
- 시간복잡도는 gpt로 분석해봐야겠

```
import java.io.*;
import java.util.*;

/*
charAt으로 하나씩 처리
문자열 리스트, 숫자 sum 둘중 하나로 보냄
문자열리스트 오름차순 정렬, 숫자는 합산해서 출력
*/
public class Main{

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        String str = br.readLine();
        int len = str.length();

        List<Character> list = new ArrayList<>();
        int sum=0;

        for(int i=0;i<len;i++) {
            char ch = str.charAt(i);

            if(ch>='0' && ch<='9') {
                sum+=ch-'0';
            }
            else {
                list.add(ch);
            }
        }

        Collections.sort(list);

        for(char ch:list) {
            bw.write(ch);
        }

        bw.write(sum+"");
        bw.flush();
        bw.close();


    }

}
```

```
/*
1칸씩자를때 길이
...
len/2칸씩 자를때 길이 
*/
class Solution {
    public int solution(String s) {
        
        int len = s.length();
        int answer = len; // 실수노트 : len이 1이면 반복문 안돌아서 이게 바로 답이 됨. maxvlue 넣으면 안됨.
        int cnt;
        String prev;
        
        
        // i : i칸씩 확인한다는 의미 
        for(int i=1;i<=len/2;i++) {
            StringBuilder sb = new StringBuilder();
            prev = s.substring(0,i);
            cnt = 1;
            
            for(int j=i;j<len;j+=i) {
                int end = Math.min(j+i,len);
                String curr = s.substring(j,end);
                
                if(curr.equals(prev)) cnt++;
                else {
                    if(cnt>1) {
                       sb.append(cnt);
                    }
                    sb.append(prev);
                    prev = curr;
                    cnt=1;
                }
            }
            if(cnt>1) {
               sb.append(cnt);
            }
            sb.append(prev);
            
            answer = Math.min(answer, sb.length());
        }
        return answer;
    }
}
```

```
/*
90도씩 돌려보면서 
상하좌우 하나씩 움직여본다
1. 90도 돌리는 메서드
2. 상하좌우 움직여보는 메서드 (메인) : 3N*3N크기 중앙에 lock
3. 정답인지 검토해주는 메서드
*/
class Solution {
    static int N;
    static int M;
    static int[][] board;
    static int[][] key;
    
    public boolean solution(int[][] key, int[][] lock) {

        N = lock.length;
        M = key.length;
        board = new int[3*N][3*N];
        Solution.key = key;
        
        
        
        // 보드 중앙에 lock 박기
        for(int i=N;i<2*N;i++) {
            for(int j=N;j<2*N;j++) {
                board[i][j] = lock[i-N][j-N];
            }
        }
        
        for(int r=0;r<4;r++) {
            if(r>0) Solution.key = rotation(); // 90도 회전
            
            for(int i=0;i<2*N;i++) {
                for(int j=0;j<2*N;j++) {
                    if(check(i,j)) return true; // 열리나 확인
                }
        }
        }
        
        
        
        return false;
    }
    
    
    // key 시작위치 input, 열리는지 output 
    static boolean check(int a, int b) {
        int[][] ch = new int[3*N][3*N];
        for(int i=N;i<2*N;i++) {
            for(int j=N;j<2*N;j++) {
                ch[i][j] += board[i][j];
            }
        }
        for(int i=0;i<M;i++) {
            for(int j=0;j<M;j++) {
                ch[i+a][j+b] += key[i][j];
            }
        }
        for(int i=N;i<2*N;i++) {
            for(int j=N;j<2*N;j++) {
                if(ch[i][j]!=1) return false;
            }
        }
        
        return true;        
    }
    
    
    static int[][] rotation() {
        int[][] rotate = new int[M][M];
        
        for(int i=0;i<M;i++) {
            for(int j=0;j<M;j++) {
                rotate[i][j] = key[j][M-i-1];
            }
        }
        
        return rotate;
    }
}
```

```
import java.io.*;
import java.util.*;

/*
1초 증가
앞방향으로 전진
벽이나 몸(2)이면 종료 (몇초인지 출력)
아니면 2생성
사과가 있으면 꼬리냅둠
사과가 없으면 꼬리 poll()해서 제거
해당 초(time)에 방향전환이 있는지 hashmap 체크
*/
public class Main{

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        int time = 0; // 흐른시간
        Map<Integer,Character> dirChange = new HashMap<>(); // 그 시각에 방향전환이 있는지 
        
        int N = Integer.parseInt(br.readLine()); // 보드크기
        int[][] board = new int[N+1][N+1]; // 맵
        int[] dr = {0,1,0,-1}; // 우하좌상
        int[] dc = {1,0,-1,0}; // (i+1)%4하면 오른쪽회전, (i+3)%4하면 왼쪽회전
        
        int K = Integer.parseInt(br.readLine()); // 사과갯수 
        StringTokenizer st;
        for(int i=0;i<K;i++) {
            st = new StringTokenizer(br.readLine());
            int r = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());
            board[r-1][c-1] = 1; // 1:사과
        }
        
        int L = Integer.parseInt(br.readLine()); // 뱀의 방향전환 횟수
        for(int i=0;i<L;i++) {
            st = new StringTokenizer(br.readLine());
            int X = Integer.parseInt(st.nextToken());
            char C = st.nextToken().charAt(0);
            dirChange.put(X,C);
        }

        int r = 0;
        int c = 0;
        int dir = 0;
        board[0][0] = 2;
        Queue<int[]> q = new ArrayDeque<>();
        q.offer(new int[]{0,0});

        while(true) {
            time++;
            
            r += dr[dir];
            c += dc[dir];

            if(r<0 || r>=N || c<0 || c>=N || board[r][c]==2) {
                bw.write(time+"");
                break;
            }

            if(board[r][c]==1) { // 사과가 있는 경우 
                board[r][c]=2;
                q.offer(new int[]{r,c});
            }
            else {
                board[r][c]=2;
                q.offer(new int[]{r,c});

                int[] tail = q.poll();
                int tr = tail[0];
                int tc = tail[1];

                board[tr][tc] = 0;
            }

            if(dirChange.containsKey(time)) {
                if(dirChange.get(time)=='D') dir = (dir+1)%4;
                else dir = (dir+3)%4;
            }

        }


        bw.flush();
        bw.close();


    }

}
```

- 시간복잡도는 O(N² + K + L + T)
