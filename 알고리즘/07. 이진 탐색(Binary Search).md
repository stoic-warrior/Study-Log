# 범위를 반씩 좁혀가는 탐색
## 순차 탐색(Sequential Search)
> 리스트 안에 특정한 데이터를 찾기 위해 앞에서부터 **순차적**으로 확인하는 방법
- 이진 탐색 전에 순차 탐색을 이해해야 한다
- 보통 정렬되지 않은 리스트에서 사용한다
- ex) 리스트에서  특정 값의 원소가 있는지 확인할 때, 리스트에서 특정 값의 원소의 수를 셀 때 count()

## 이진탐색 : 반으로 쪼개면서 탐색하기
> 찾으려는 데이터와 중간점 데이터를 반복적으로 비교
- 사용하는 변수 (3개)
  - 시작점(Start): 탐색 범위의 맨 앞 인덱스
  - 끝점(End): 탐색 범위의 맨 뒤 인덱스
  - 중간점(Mid): 시작점과 끝점의 중간 인덱스 ($$(시작점 + 끝점) / 2$$, 소수점 이하 버림)
- 정렬된 데이터에만 사용
- <img width="540" height="319" alt="image" src="https://github.com/user-attachments/assets/adbc3228-07f2-4122-a7eb-0dc561e7bf22" />

```
public static int binarySearch(int[] array, int target, int start, int end) {
        while (start <= end) {
            int mid = (start + end) / 2;

            // 찾은 경우 중간점 인덱스 반환
            if (array[mid] == target) {
                return mid;
            }
            // 중간점의 값보다 찾고자 하는 값이 작은 경우 왼쪽 확인
            else if (array[mid] > target) {
                end = mid - 1;
            }
            // 중간점의 값보다 찾고자 하는 값이 큰 경우 오른쪽 확인
            else {
                start = mid + 1;
            }
        }
        return -1; // 찾지 못한 경우
    }
```

- 시간 복잡도: $O(\log N)$
- 밑은 2이다. 단계를 거칠때마다 평균적으로 절반씩 줄어들기 때문.
- 실전 팁
  - 코딩 테스트에서 탐색 범위가 2,000만을 넘어가거나, 데이터 개수가 1,000만 단위 이상이라면 반드시 이진 탐색을 떠올려야 함
  - 이진 탐색 코드는 눈 감고도 짤 수 있을 정도로 외워두는 것이 좋다. 코딩 테스트 단골 문제일 뿐만 아니라, 다른 복잡한 알고리즘(그리디, 파라메트릭 서치 등)과 결합되어 자주 나오기 때문
## 트리 자료구조
- 이진 탐색의 원리를 데이터베이스나 파일 시스템처럼 거대한 규모에서 활용하기 위해 사용하는 것이 바로 트리(Tree) 자료구조 (정렬된 상태를 유지)
- 트리는 노드(Node)들이 부모-자식 관계로 연결된 계층적 구조
  - 노드(Node): 정보를 담고 있는 기본 단위입니다.
  - 루트 노드(Root Node): 트리의 가장 최상단에 있는 시작 노드입니다.
  - 단말 노드(Leaf Node): 자식이 없는, 트리의 가장 끝에 위치한 노드입니다.
  - 서브 트리(Subtree): 큰 트리 안에 포함된 작은 트리 구조를 말합니다.

public class Main{

    static int N;
    static int[] parts;

    static boolean binarySearch(int target) {
        int left = 0;
        int right = N-1;
        

        while(left<=right) {
            int mid = left + (right - left)/2;

            if(parts[mid]==target) {
                return true;
            }
            else if(parts[mid]<target) {
                left = mid+1;
            }
            else {
                right = mid-1;
            }
        }

        return false;

    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        
        N = Integer.parseInt(br.readLine());
        StringTokenizer st = new StringTokenizer(br.readLine());
        parts = new int[N];
        for(int i=0;i<N;i++) {
            parts[i] = Integer.parseInt(st.nextToken());
        }
        Arrays.sort(parts);

        int M = Integer.parseInt(br.readLine());
        st = new StringTokenizer(br.readLine());
        for(int i=0;i<M;i++) {
            boolean isHave = binarySearch(Integer.parseInt(st.nextToken()));

            if(isHave) bw.write("yes");
            else bw.write("no");

            if(i != M-1) bw.write(" ");
        }
        

        
        bw.flush();
        bw.close();


    }

    
}
```

- 시간복잡도 : $O((N+M) \log N)$
  - 정렬시간, 이진탐색 M번 시간

```
import java.io.*;
import java.util.*;


// 이진탐색으로 최대높이 찾기 
public class Main{

    static int N;
    static int max;
    static int min;
    static int[] rice;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
      
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken());
        int M = Integer.parseInt(st.nextToken());

        rice = new int[N];
        st = new StringTokenizer(br.readLine());
        for(int i=0;i<N;i++) {
            rice[i] = Integer.parseInt(st.nextToken());
        }

        max = 0;
        min = Integer.MAX_VALUE;
        for(int i=0;i<N;i++) {
            if(rice[i]>max) max = rice[i];
            if(rice[i]<min) min = rice[i];
        }

        int h = bs(M);

        bw.write(h + "");
        bw.flush();
        bw.close();


    }

    // 손님이 필요한양 input, 높이이 최대값 output
    static int bs(int M) {
        int left = min;
        int right = max;

        while(left<=right) {
            int mid = left + (right-left)/2;
            int len = cutLen(mid);

            if(len == M) {
                return mid;
            }
            else if(len < M) {
                right = mid-1;
            }
            else {
                left = mid+1;
            }
        }

        return right;
    }

    // 높이input, 썰린떡 output
    static int cutLen(int mid) {
        int sum = 0;
        for(int i=0;i<N;i++) {
            int len = rice[i] - mid;
            if(len>0) sum+=len;
        }

        return sum;
    }
}
