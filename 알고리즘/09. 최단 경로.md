# 가장 빠른 길 찾기
- 최단 경로는 상황에 맞는 효율적 알고리즘이 이미 정립되어 있다
  - ex) 한 지점에서 다른 특정 지점까지 최단 거리
  - ex) 모든 지점에서 다른 모든 지점까지 최단 거리
- 최단 경로 문제는 보통 그래프를 통해 출제
- <img width="304" height="193" alt="image" src="https://github.com/user-attachments/assets/d4adcce8-963f-410d-86d8-26d89d13e593" />

  - 노드
  - 간선
  - 가중치
- 학부 수준 최단거리 3대 알고리즘
  - 다익스트라
  - 플로이드 워셜
  - 벨만포드

| 알고리즘 | 특징 | 알고리즘 유형 |
| :--- | :--- | :--- |
| **다익스트라 (Dijkstra)** | **한 지점**에서 다른 모든 지점까지의 최단 거리를 구할 때 사용 | **그리디 (Greedy)** |
| **플로이드 워셜 (Floyd-Warshall)** | **모든 지점**에서 다른 모든 지점까지의 최단 거리를 모두 구할 때 사용 | **다이나믹 프로그래밍 (DP)** |
| **벨만 포드 (Bellman-Ford)** | 간선의 가중치가 **음수**일 때도 사용 가능 (이 책에선 생략됨) | - |
