# 다익스트라 (Dijkstra)
> 특정 노드에서 출발하여 다른 모든 노드로 가는 각각의 최단 경로를 구하는 알고리즘
- **특징:** '음의 간선'(0보다 작은 값을 가지는 간선)이 없을 때 정상적으로 동작합니다.
- **유형:** 매번 '가장 비용이 적은 노드'를 선택하므로 **그리디(Greedy) 알고리즘**으로 분류됩니다.
- 알고리즘 대회를 준비한다면 다익스트라는 자다가도 일어나서 구현할 정도로 숙달
- 구현 방법:
  - 구현하기 쉽지만 느린 코드 (선형 탐색)
  - 구현하기 까다롭지만 빠른 코드 (우선순위 큐/힙 사용) -> 실전 코딩 테스트에서는 이 방법을 숙달해야 합니다.
## 동작 원리 (6단계)
1. 출발 노드 설정: 시작 지점을 정합니다.
2. 최단 거리 테이블 초기화: 출발 노드는 0, 나머지 노드는 무한(INF)으로 설정합니다.
3. 노드 선택: 방문하지 않은 노드 중에서 최단 거리가 가장 짧은 노드를 선택합니다.
4. 테이블 갱신: 해당 노드를 거쳐 다른 노드로 가는 비용을 계산하여 기존 값보다 작으면 교체합니다.
5. 반복: 모든 노드를 방문할 때까지 3번과 4번 과정을 반복합니다.
- 최단 거리 테이블 : 각 노드에 대한 현재까지의 최단 거리
- 한 단계당 하나의 노드 최단 거리 확정 : 3번과정에서 선택된건 최단거리가 확정이다. (그래서 마지막 노드는 4번이 필요없다. 모든 노드 확정이라서)

## 방법1
> 매 단계마다 '방문하지 않은 노드 중 최단 거리가 가장 짧은 노드'를 찾기 위해 1차원 리스트의 모든 원소를 순차 탐색
- 핵심 로직
  - Node 클래스: 연결된 노드와 가중치를 저장하는 객체입니다.
  - getSmallestNode(): visited되지 않은 노드 중 최단 거리 값이 가장 작은 인덱스를 찾는 $O(V)$ 함수입니다.
  - dijkstra(): 시작 노드부터 차례대로 모든 노드를 탐색하며 테이블을 갱신합니다
```
import java.io.*;
import java.util.*;

public class Main {

    static final int INF = (int) 1e9;

    static int n, m, start;
    static ArrayList<ArrayList<Node>> graph = new ArrayList<>();
    static boolean[] visited;
    static int[] distance;

    static class Node {
        int index;
        int cost;

        Node(int index, int cost) {
            this.index = index;
            this.cost = cost;
        }
    }

    // 방문하지 않은 노드 중에서 최단 거리가 가장 짧은 노드 번호 반환
    static int getSmallestNode() {
        int minValue = INF;
        int index = 0;

        for (int i = 1; i <= n; i++) {
            if (!visited[i] && distance[i] < minValue) {
                minValue = distance[i];
                index = i;
            }
        }

        return index;
    }

    static void dijkstra(int start) {
        // 시작 노드 초기화
        distance[start] = 0;
        visited[start] = true;

        // 시작 노드와 연결된 노드들의 거리 초기화
        for (Node node : graph.get(start)) {
            distance[node.index] = node.cost;
        }

        // 시작 노드를 제외한 전체 n - 1개의 노드 처리
        for (int i = 0; i < n - 1; i++) {
            int now = getSmallestNode();

            // 더 이상 도달 가능한 노드가 없으면 종료
            if (now == 0) break;

            visited[now] = true;

            // 현재 노드와 연결된 다른 노드 확인
            for (Node node : graph.get(now)) {
                int cost = distance[now] + node.cost;

                // 현재 노드를 거쳐서 가는 거리가 더 짧으면 갱신
                if (cost < distance[node.index]) {
                    distance[node.index] = cost;
                }
            }
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken()); // 노드 개수
        m = Integer.parseInt(st.nextToken()); // 간선 개수

        start = Integer.parseInt(br.readLine());

        visited = new boolean[n + 1];
        distance = new int[n + 1];

        Arrays.fill(distance, INF);

        // 그래프 초기화
        for (int i = 0; i <= n; i++) {
            graph.add(new ArrayList<>());
        }

        // 간선 정보 입력
        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // a번 노드에서 b번 노드로 가는 비용 c
            graph.get(a).add(new Node(b, c));
        }

        dijkstra(start);

        StringBuilder sb = new StringBuilder();

        for (int i = 1; i <= n; i++) {
            if (distance[i] == INF) {
                sb.append("INFINITY").append('\n');
            } else {
                sb.append(distance[i]).append('\n');
            }
        }

        System.out.print(sb);
    }
}
```
- 시간복잡도 : O(V^2 + E)
  - V개 탐색하는 순차탐색을 V번 호출
  - 간선 E개 탐색
  - E는 대부분 V^2보다 작아서 O(V^2)라고도 함
## 방법2
> 선형탐색 대신 우선순위 큐(heap) 활용
- 우선순위 큐 : 우선순위가 가장 높은 데이터를 가장 먼저 삭제, 힙으로 구현

```
import java.io.*;
import java.util.*;

public class Main {

    static final int INF = (int) 1e9;

    static int n, m, start;
    static ArrayList<ArrayList<Node>> graph = new ArrayList<>();
    static int[] distance;

    static class Node implements Comparable<Node> {
        int index; // 노드 번호
        int dist;  // 시작 노드에서 이 노드까지의 거리

        Node(int index, int dist) {
            this.index = index;
            this.dist = dist;
        }

        // 거리가 짧은 노드가 우선순위 큐에서 먼저 나오게 함
        @Override
        public int compareTo(Node other) {
            return Integer.compare(this.dist, other.dist);
        }
    }

    static void dijkstra(int start) {
        PriorityQueue<Node> pq = new PriorityQueue<>();

        // 시작 노드까지의 거리는 0
        distance[start] = 0;
        pq.offer(new Node(start, 0));

        while (!pq.isEmpty()) {
            Node current = pq.poll();

            int now = current.index;
            int dist = current.dist;

            // 이미 더 짧은 경로가 발견된 적 있다면 무시
            if (distance[now] < dist) {
                continue;
            }

            // 현재 노드와 연결된 다른 노드들 확인
            for (Node next : graph.get(now)) {
                int cost = dist + next.dist;

                // 현재 노드를 거쳐서 가는 거리가 더 짧으면 갱신
                if (cost < distance[next.index]) {
                    distance[next.index] = cost;
                    pq.offer(new Node(next.index, cost));
                }
            }
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());

        n = Integer.parseInt(st.nextToken()); // 노드 개수
        m = Integer.parseInt(st.nextToken()); // 간선 개수

        start = Integer.parseInt(br.readLine());

        distance = new int[n + 1];
        Arrays.fill(distance, INF);

        for (int i = 0; i <= n; i++) {
            graph.add(new ArrayList<>());
        }

        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // a번 노드에서 b번 노드로 가는 비용이 c
            graph.get(a).add(new Node(b, c));
        }

        dijkstra(start);

        StringBuilder sb = new StringBuilder();

        for (int i = 1; i <= n; i++) {
            if (distance[i] == INF) {
                sb.append("INFINITY").append('\n');
            } else {
                sb.append(distance[i]).append('\n');
            }
        }

        System.out.print(sb);
    }
}
```
# 플로이드 워셜
> 모든 노드 → 모든 노드 최단거리
## 핵심 아이디어: "거쳐가는 노드"
- "A에서 B로 직접 가는 것보다, 중간에 K를 거쳐서 가는 게 더 빠를까?" 를 반복
- $$D_{ab} = \min(D_{ab}, D_{ak} + D_{kb})$$
  - $D_{ab}$: 현재까지 알고 있는 $a$에서 $b$로 가는 최단 거리
  - $D_{ak} + D_{kb}$: $k$번 노드를 거쳐서 가는 거리
## 알고리즘의 특징
- 다이나믹 프로그래밍 (DP): 매 단계마다 특정 노드를 거쳐가는 경우를 고려하여 테이블을 갱신하므로 DP에 해당합니다.
- 2차원 리스트 사용: 모든 노드 쌍에 대한 거리를 저장해야 하므로 $N \times N$ 크기의 테이블이 필요합니다.
- 시간 복잡도 $O(N^3)$: 노드의 개수가 $N$개일 때, 거쳐가는 노드를 기준으로 $N$번 반복하며, 각 단계마다 $O(N^2)$의 연산(2차원 테이블 갱신)을 수행하기 때문입니다
```
import java.io.*;
import java.util.*;

public class Main {

    static final int INF = (int) 1e9;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();

        int n = Integer.parseInt(br.readLine()); // 노드 개수
        int m = Integer.parseInt(br.readLine()); // 간선 개수

        int[][] graph = new int[n + 1][n + 1];

        // 모든 거리를 무한으로 초기화
        for (int i = 1; i <= n; i++) {
            Arrays.fill(graph[i], INF);
        }

        // 자기 자신으로 가는 비용은 0
        for (int i = 1; i <= n; i++) {
            graph[i][i] = 0;
        }

        // 간선 정보 입력
        for (int i = 0; i < m; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // a에서 b로 가는 비용 c
            // 같은 간선이 여러 번 들어올 수 있으므로 더 작은 값 저장
            graph[a][b] = Math.min(graph[a][b], c);
        }

        // 플로이드 워셜 수행
        // k = 거쳐 가는 노드
        for (int k = 1; k <= n; k++) {
            // a = 출발 노드
            for (int a = 1; a <= n; a++) {
                // b = 도착 노드
                for (int b = 1; b <= n; b++) {
                    graph[a][b] = Math.min(graph[a][b], graph[a][k] + graph[k][b]);
                }
            }
        }

        // 결과 출력
        for (int a = 1; a <= n; a++) {
            for (int b = 1; b <= n; b++) {
                if (graph[a][b] == INF) {
                    sb.append("INFINITY").append(' ');
                } else {
                    sb.append(graph[a][b]).append(' ');
                }
            }
            sb.append('\n');
        }

        System.out.print(sb);
    }
}
```

- **다익스트라 vs 플로이드 워셜**:
  - **다익스트라:** 한 지점에서 다른 모든 지점 ($1 \to N$), $1$차원 리스트 사용.
  - **플로이드 워셜:** 모든 지점에서 모든 지점 ($N \to N$), $2$차원 리스트 사용.
- **구현 시 주의점:** 3중 for문을 사용하며, 가장 바깥쪽 for문이 '거쳐가는 노드(K)'가 되어야 합니다.

## A) 플로이드 워
```
import java.io.*;
import java.util.*;

// 플로이드 워셜
// ab = min(ab, akkb)
public class Main{
    static final int INF = (int) 1e9; // 실수노트 : MAX_VALUE로 해버리면 더할때 오버플로우
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken()); // 노드
        int M = Integer.parseInt(st.nextToken()); // 엣지

        int[][] graph = new int[N+1][N+1];

        for(int i=1;i<=N;i++) {
            for(int j=1;j<=N;j++) {
                graph[i][j] = INF;
            }
        }

        for(int i=1;i<=N;i++) {
            graph[i][i] = 0;
        }

        for(int i=0;i<M;i++) {
            st = new StringTokenizer(br.readLine());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            graph[a][b] = 1;
            graph[b][a] = 1;
        }

        for(int k=1;k<=N;k++) {
            for(int i=1;i<=N;i++) {
                for(int j=1;j<=N;j++) {
                    graph[i][j] = Math.min(graph[i][j], graph[i][k]+graph[k][j]);
                }
            }
        }

        st = new StringTokenizer(br.readLine());
        int X = Integer.parseInt(st.nextToken());
        int K = Integer.parseInt(st.nextToken());

        int result = graph[1][K]+graph[K][X];

        bw.write(result>=INF?"-1":result + "");     // 실수노트 : ==INF가 아니라 >=INF로 검사해야 함
        bw.flush();
        bw.close();


    }

}
```

- 시간복잡도 : O(N^3)
- N<=100 이라서 쓸 수 있음

## A) 다익스트라
```
import java.io.*;
import java.util.*;

// 다익스트라 실행
// 그 후 INF아닌 노드 갯수, 거리 최대값
public class Main{
    static final int INF = (int) 1e9; 
    static int[] dist;
    static ArrayList<Node>[] adj;

    static class Node implements Comparable<Node> {
        int v;
        int w;

        Node(int v,int w) {
            this.v=v;
            this.w=w;
        }

        @Override
        public int compareTo(Node other) {
            return Integer.compare(this.w,other.w);
        }
    }
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken()); // 노드
        int M = Integer.parseInt(st.nextToken()); // 엣지
        int C = Integer.parseInt(st.nextToken()); // 출발지 

        // 초기화
        adj = new ArrayList[N+1];
        dist = new int[N+1];
        for(int i=1;i<=N;i++) {
            adj[i] = new ArrayList<>();
        }
        Arrays.fill(dist,INF);

        for(int i=0;i<M;i++) {
            st = new StringTokenizer(br.readLine());
            int X = Integer.parseInt(st.nextToken()); // 출발
            int Y = Integer.parseInt(st.nextToken()); // 도착
            int Z = Integer.parseInt(st.nextToken()); // 거리 

            adj[X].add(new Node(Y,Z));
        }

        dijkstra(C);

        int cnt=0; 
        int max=0;
        for(int i=1;i<=N;i++) {
            if(dist[i]!=INF && dist[i]!=0) { // 출발지는 빼야함
                cnt++;
                max = Math.max(max,dist[i]);
            }
        }

        bw.write(cnt+" "+max);   
        bw.flush();
        bw.close();


    }


    static void dijkstra(int start) {
        PriorityQueue<Node> pq = new PriorityQueue<>();
        dist[start] = 0;
        pq.offer(new Node(start,0)); // 여기에 넣는 w는 dist이다. adj의 w와 헷깔리지 말자

        while(!pq.isEmpty()) {
            Node node = pq.poll();
            int u = node.v;
            int w = node.w;

            if(dist[u]<w) continue;

            for(Node next:adj[u]) {
                if(dist[next.v]>dist[u]+next.w) {
                    dist[next.v] = dist[u] + next.w;
                    pq.offer(new Node(next.v,dist[next.v]));
                }
            }
        }
    }

}
```
