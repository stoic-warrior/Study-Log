# 다익스트라 (Dijkstra)
> 특정 노드에서 출발하여 다른 모든 노드로 가는 각각의 최단 경로를 구하는 알고리즘
- **특징:** '음의 간선'(0보다 작은 값을 가지는 간선)이 없을 때 정상적으로 동작합니다.
- **유형:** 매번 '가장 비용이 적은 노드'를 선택하므로 **그리디(Greedy) 알고리즘**으로 분류됩니다.
- 알고리즘 대회를 준비한다면 다익스트라는 자다가도 일어나서 구현할 정도로 숙달
- 구현 방법:
  - 구현하기 쉽지만 느린 코드 (선형 탐색)
  - 구현하기 까다롭지만 빠른 코드 (우선순위 큐/힙 사용) -> 실전 코딩 테스트에서는 이 방법을 숙달해야 합니다.
## 동작 원리 (6단계)
1. 출발 노드 설정: 시작 지점을 정합니다.
2. 최단 거리 테이블 초기화: 출발 노드는 0, 나머지 노드는 무한(INF)으로 설정합니다.
3. 노드 선택: 방문하지 않은 노드 중에서 최단 거리가 가장 짧은 노드를 선택합니다.
4. 테이블 갱신: 해당 노드를 거쳐 다른 노드로 가는 비용을 계산하여 기존 값보다 작으면 교체합니다.
5. 반복: 모든 노드를 방문할 때까지 3번과 4번 과정을 반복합니다.
- 최단 거리 테이블 : 각 노드에 대한 현재까지의 최단 거리
- 한 단계당 하나의 노드 최단 거리 확정 : 3번과정에서 선택된건 최단거리가 확정이다. (그래서 마지막 노드는 4번이 필요없다. 모든 노드 확정이라서)

## 방법1
> 매 단계마다 '방문하지 않은 노드 중 최단 거리가 가장 짧은 노드'를 찾기 위해 1차원 리스트의 모든 원소를 순차 탐색
- 핵심 로직
  - Node 클래스: 연결된 노드와 가중치를 저장하는 객체입니다.
  - getSmallestNode(): visited되지 않은 노드 중 최단 거리 값이 가장 작은 인덱스를 찾는 $O(V)$ 함수입니다.
  - dijkstra(): 시작 노드부터 차례대로 모든 노드를 탐색하며 테이블을 갱신합니다
```
import java.io.*;
import java.util.*;

public class Main {

    static final int INF = (int) 1e9;

    static int n, m, start;
    static ArrayList<ArrayList<Node>> graph = new ArrayList<>();
    static boolean[] visited;
    static int[] distance;

    static class Node {
        int index;
        int cost;

        Node(int index, int cost) {
            this.index = index;
            this.cost = cost;
        }
    }

    // 방문하지 않은 노드 중에서 최단 거리가 가장 짧은 노드 번호 반환
    static int getSmallestNode() {
        int minValue = INF;
        int index = 0;

        for (int i = 1; i <= n; i++) {
            if (!visited[i] && distance[i] < minValue) {
                minValue = distance[i];
                index = i;
            }
        }

        return index;
    }

    static void dijkstra(int start) {
        // 시작 노드 초기화
        distance[start] = 0;
        visited[start] = true;

        // 시작 노드와 연결된 노드들의 거리 초기화
        for (Node node : graph.get(start)) {
            distance[node.index] = node.cost;
        }

        // 시작 노드를 제외한 전체 n - 1개의 노드 처리
        for (int i = 0; i < n - 1; i++) {
            int now = getSmallestNode();

            // 더 이상 도달 가능한 노드가 없으면 종료
            if (now == 0) break;

            visited[now] = true;

            // 현재 노드와 연결된 다른 노드 확인
            for (Node node : graph.get(now)) {
                int cost = distance[now] + node.cost;

                // 현재 노드를 거쳐서 가는 거리가 더 짧으면 갱신
                if (cost < distance[node.index]) {
                    distance[node.index] = cost;
                }
            }
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken()); // 노드 개수
        m = Integer.parseInt(st.nextToken()); // 간선 개수

        start = Integer.parseInt(br.readLine());

        visited = new boolean[n + 1];
        distance = new int[n + 1];

        Arrays.fill(distance, INF);

        // 그래프 초기화
        for (int i = 0; i <= n; i++) {
            graph.add(new ArrayList<>());
        }

        // 간선 정보 입력
        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // a번 노드에서 b번 노드로 가는 비용 c
            graph.get(a).add(new Node(b, c));
        }

        dijkstra(start);

        StringBuilder sb = new StringBuilder();

        for (int i = 1; i <= n; i++) {
            if (distance[i] == INF) {
                sb.append("INFINITY").append('\n');
            } else {
                sb.append(distance[i]).append('\n');
            }
        }

        System.out.print(sb);
    }
}
```
- 시간복잡도 : O(V^2 + E)
  - V개 탐색하는 순차탐색을 V번 호출
  - 간선 E개 탐색
  - E는 대부분 V^2보다 작아서 O(V^2)라고도 함
## 방법2
> 선형탐색 대신 우선순위 큐(heap) 활용
- 우선순위 큐 : 우선순위가 가장 높은 데이터를 가장 먼저 삭제, 힙으로 구현

- 시간복잡도 : O(E log V)
```
import java.io.*;
import java.util.*;

public class Main {

    static final int INF = (int) 1e9;

    static int n, m, start;
    static ArrayList<ArrayList<Node>> graph = new ArrayList<>();
    static int[] distance;

    static class Node implements Comparable<Node> {
        int index; // 노드 번호
        int dist;  // 시작 노드에서 이 노드까지의 거리

        Node(int index, int dist) {
            this.index = index;
            this.dist = dist;
        }

        // 거리가 짧은 노드가 우선순위 큐에서 먼저 나오게 함
        @Override
        public int compareTo(Node other) {
            return Integer.compare(this.dist, other.dist);
        }
    }

    static void dijkstra(int start) {
        PriorityQueue<Node> pq = new PriorityQueue<>();

        // 시작 노드까지의 거리는 0
        distance[start] = 0;
        pq.offer(new Node(start, 0));

        while (!pq.isEmpty()) {
            Node current = pq.poll();

            int now = current.index;
            int dist = current.dist;

            // 이미 더 짧은 경로가 발견된 적 있다면 무시
            if (distance[now] < dist) {
                continue;
            }

            // 현재 노드와 연결된 다른 노드들 확인
            for (Node next : graph.get(now)) {
                int cost = dist + next.dist;

                // 현재 노드를 거쳐서 가는 거리가 더 짧으면 갱신
                if (cost < distance[next.index]) {
                    distance[next.index] = cost;
                    pq.offer(new Node(next.index, cost));
                }
            }
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());

        n = Integer.parseInt(st.nextToken()); // 노드 개수
        m = Integer.parseInt(st.nextToken()); // 간선 개수

        start = Integer.parseInt(br.readLine());

        distance = new int[n + 1];
        Arrays.fill(distance, INF);

        for (int i = 0; i <= n; i++) {
            graph.add(new ArrayList<>());
        }

        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int c = Integer.parseInt(st.nextToken());

            // a번 노드에서 b번 노드로 가는 비용이 c
            graph.get(a).add(new Node(b, c));
        }

        dijkstra(start);

        StringBuilder sb = new StringBuilder();

        for (int i = 1; i <= n; i++) {
            if (distance[i] == INF) {
                sb.append("INFINITY").append('\n');
            } else {
                sb.append(distance[i]).append('\n');
            }
        }

        System.out.print(sb);
    }
}
```
