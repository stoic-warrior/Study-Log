# 꼭 필요한 자료구조 기초
- **탐색** : 많은 양의 데이터 중에서 원하는 데이터를 찾는 과정
  - 그래프, 트리 등에서의 탐색 문제를 자주 다룸
  - 대표적인 탐색 알고리즘이 DFS,BFS
  - DFS, BFS를 제대로 이해하려면 스택과 큐같은 자료구조에 대한 이해가 있어야 함 
- **자료구조** : 데이터를 표현하고 관리하고 처리하기 위한 구조
- 스택과 큐의 핵심함수
  - 삽입(Push): 데이터를 넣는 작업
  - 삭제(Pop): 데이터를 꺼내는 작업
- 주의 
  - 오버플로(Overflow): 이미 꽉 찬 자료구조에 데이터를 더 넣으려고 할 때 발생합니다. (넘침)
  - 언더플로(Underflow): 데이터가 하나도 없는데 삭제 연산을 수행하려고 할 때 발생합니다. (비어있음)
