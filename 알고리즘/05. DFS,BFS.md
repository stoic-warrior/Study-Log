# 꼭 필요한 자료구조 기초
- **탐색** : 많은 양의 데이터 중에서 원하는 데이터를 찾는 과정
  - 그래프, 트리 등에서의 탐색 문제를 자주 다룸
  - 대표적인 탐색 알고리즘이 DFS,BFS
  - DFS, BFS를 제대로 이해하려면 스택과 큐같은 자료구조에 대한 이해가 있어야 함 
- **자료구조** : 데이터를 표현하고 관리하고 처리하기 위한 구조
- **스택과 큐의 핵심함수**
  - **삽입(Push)**: 데이터를 넣는 작업
  - **삭제(Pop)**: 데이터를 꺼내는 작업
- **주의**
  - **오버플로(Overflow)**: 이미 꽉 찬 자료구조에 데이터를 더 넣으려고 할 때 발생합니다. (넘침)
  - **언더플로(Underflow)**: 데이터가 하나도 없는데 삭제 연산을 수행하려고 할 때 발생합니다. (비어있음)
## 스택
> 입구와 출구가 하나인 원통
<img width="132" height="188" alt="image" src="https://github.com/user-attachments/assets/ceb26408-e3cd-45ad-945e-831ab500b79a" />

- **LIFO (Last-In First-Out)**: 나중에 들어온 데이터가 가장 먼저 나가는 구조입니다.
- **FILO (First-In Last-Out)**: 반대로 먼저 들어온 데이터는 가장 나중에 나가게 됩니다.
- 스택은 **뒤로 가기, 실행 취소(Ctrl+Z), 함수 호출 기록** 등에 사용

```
import java.util.ArrayDeque;
import java.util.Deque;

public class Main {
    public static void main(String[] args) {
        // Deque 인터페이스를 통해 ArrayDeque 생성
        Deque<Integer> stack = new ArrayDeque<>();

        // 1. 삽입 (Push)
        stack.push(5);
        stack.push(2);
        stack.push(3);
        stack.push(7);

        // 2. 최상단 제거 (Pop)
        stack.pop(); // 7이 나감

        stack.push(1);
        stack.push(4);

        // 3. 최상단 제거 (Pop)
        stack.pop(); // 4가 나감

        // 결과 확인
        System.out.println(stack); // (상단부터 출력됨)
        
        // 4. 맨 위의 값만 확인 (Peek)
        System.out.println(stack.peek()); // 1
    }
}
```
- **Push**: `addLast()` 또는 `push()`
- **Pop**: `removeLast()` 또는 `pop()`
- Java 공식 문서에서는 `Stack` 클래스보다 `ArrayDeque`를 사용하는 것을 더 권장 (stack은 옛날에 설계되어 성능과 안정성 둘다 낮음)
- `Deque<Integer>`처럼 스택에 들어갈 데이터의 타입을 미리 정해야 함 (제네릭)

- 컴퓨터 내부에서 재귀함수는 스택 자료구조를 이용 (메모리의 스택공간) 
- 종료조건이 없으면, StackOverflowError가 발생하며 프로그램이 강제로 종료
### 반드시 필요한 '종료 조건'
```
public class RecursiveExitExample {
    public static void recursiveFunction(int i) {
        // 100번째 호출되었을 때 종료되도록 조건 명시 (Base Case)
        if (i == 100) {
            return;
        }

        System.out.println(i + "번째 재귀 함수에서 " + (i + 1) + "번째 재귀 함수를 호출합니다.");
        
        recursiveFunction(i + 1);
        
        System.out.println(i + "번째 재귀 함수를 종료합니다.");
    }

    public static void main(String[] args) {
        // 1부터 시작하여 재귀 함수 호출
        recursiveFunction(1);
    }
}
```

- 위의 if문 처럼, 재귀함수는 종료조건이 반드시 필요 (무한루프 방지)
### 팩토리얼 예제
```
public class FactorialExample {

    // 1. 반복적으로 구현한 n!
    public static int factorialIterative(int n) {
        int result = 1;
        // 1부터 n까지의 수를 차례대로 곱하기
        for (int i = 1; i <= n; i++) {
            result *= i;
        }
        return result;
    }

    // 2. 재귀적으로 구현한 n!
    public static int factorialRecursive(int n) {
        // n이 1 이하인 경우 1을 반환 (종료 조건)
        if (n <= 1) {
            return 1;
        }
        // n! = n * (n - 1)! 공식 그대로 작성
        return n * factorialRecursive(n - 1);
    }

    public static void main(String[] args) {
        // 각각의 방식으로 구현한 5! 출력
        System.out.println("반복적으로 구현: " + factorialIterative(5));
        System.out.println("재귀적으로 구현: " + factorialRecursive(5));
    }
}
```

- 위 반복문과 재귀 함수의 결과는 같다
- 재귀 함수의 장점은 코드가 간결해진다는 것이다 (위 예시에선 아니지만..)
- 재귀 함수는 수학의 **점화식**을 활용한다 (점화식은 DP에서도 쓰인다)

## DFS
> 스택or 재귀 함수로 구현
- 1️⃣ 시작 노드를 스택에 넣는다.
- 2️⃣ 스택이 빌 때까지 반복한다.
  - 1️⃣ 스택에서 노드를 하나 꺼낸다. (pop)
    - 1 이미 방문한 노드라면 무시한다.
    - 2 방문하지 않은 노드라면:
      - 방문 처리한다.
      - 해당 노드를 출력/처리한다.
      - 인접 노드 중 방문하지 않은 노드를 스택에 넣는다.
```
        // 1️⃣ 시작 노드를 스택에 넣는다.
        stack.push(1);

        // 2️⃣ 스택이 빌 때까지 반복
        while (!stack.isEmpty()) {

            // 1️⃣ 스택에서 노드 꺼내기
            int now = stack.pop();

            // 1️⃣-1 이미 방문했으면 무시
            if (visited[now]) continue;

            // 1️⃣-2 방문하지 않았다면
            visited[now] = true;              // 방문 처리
            System.out.print(now + " ");      // 출력

            // 인접 노드 push
            for (int next : graph[now]) {
                if (!visited[next]) {
                    stack.push(next);
                }
            }
        }
```
## BFS
> 큐로 구현
- 시작 노드를 큐에 넣고 방문 처리한다.
- 큐가 빌 때까지 반복한다.
  - 큐에서 노드를 하나 꺼낸다. (poll)
  - 해당 노드를 출력/처리한다.
  - 인접 노드 중 방문하지 않은 노드를:
    - 큐에 넣는다.
    - 방문 처리한다.
```
        // 1️⃣ 시작 노드를 큐에 넣고 방문 처리
        queue.offer(1);
        visited[1] = true;

        // 2️⃣ 큐가 빌 때까지 반복
        while (!queue.isEmpty()) {

            // 2-1️⃣ 큐에서 노드 꺼내기
            int now = queue.poll();

            // 2-2️⃣ 출력
            System.out.print(now + " ");

            // 2-3️⃣ 인접 노드 처리
            for (int next : graph[now]) {
                if (!visited[next]) {
                    queue.offer(next);     // 큐에 넣기
                    visited[next] = true;  // 방문 처리 (여기서!)
                }
            }
        }
    }
```

- 큐에 넣을때 방문처리 하는 이유는, BFS 구조상 그렇게 해야 중복push를 막을 수 있기 때문이다

```
/*
그래프 노드 완탐
0인데 방문x일때 dfs 호출하고 cnt++
dfs함수는 dfs로 방문표시

public class Main{

    static int[][] graph;
    static  boolean[][] visited;
    static int M;
    static int N;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken()); // 세로길이
        M = Integer.parseInt(st.nextToken()); // 가로길이

        graph = new int[N][M];
        visited = new boolean[N][M];
        int cnt=0; // 아이스크림 개수

        for(int i=0;i<N;i++) {
            String line = br.readLine();
            for(int j=0;j<M;j++) {
                graph[i][j] = line.charAt(j)- '0'; // char에서 int로 바꾸는 테크닉
            }
        }

        for(int i=0;i<N;i++) {
            for(int j=0;j<M;j++) {
                if(!visited[i][j] && graph[i][j]==0) { // graph[i][j]==0를 놓쳤다..
                    cnt++;
                    dfs(i,j);
                }
            }
        }


        bw.write(cnt +" ");
        bw.flush();
        bw.close();


    }

    // 미방문시 방문체크 후 인접한곳 재귀
    static void dfs(int i, int j) {
        if(i<0 || i>N-1 || j<0 || j>M-1) return;
        if(graph[i][j]==1) return;
        if(!visited[i][j]) {
            visited[i][j] = true;
            dfs(i-1,j);
            dfs(i+1,j);
            dfs(i,j-1);
            dfs(i,j+1);
        }
        
    }

    
}
```

- 실수노트 : graph[i][j]==0를 놓쳤다..

### A)
- 최단거리는 BFS로 풀어야 한다
```
public class Main{

    static int[][] graph;
    static  boolean[][] visited;
    static int M;
    static int N;
    static int[] dx = {-1,1,0,0};
    static int[] dy = {0,0,-1,1};
    static int[][] cnt;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken()); // 세로길이
        M = Integer.parseInt(st.nextToken()); // 가로길이

        graph = new int[N][M];
        visited = new boolean[N][M];
        cnt = new int[N][M];

        for(int i=0;i<N;i++) {
            String line = br.readLine();
            for(int j=0;j<M;j++) {
                graph[i][j] = line.charAt(j)- '0'; // char에서 int로 바꾸는 테크닉
            }
        }

        int result = bfs();

        bw.write(result +" ");
        bw.flush();
        bw.close();


    }

    static int bfs() {
        Queue<int[]> q = new ArrayDeque<>();

        q.offer(new int[]{0,0});
        visited[0][0] = true;
        cnt[0][0] = 1;

        while(!q.isEmpty()) {
            int[] node = q.poll();
            int x = node[0];
            int y = node[1];
            

            for(int i=0;i<4;i++) {
                int nx = x + dx[i];
                int ny = y + dy[i];

                if(nx<0 || nx>N-1 || ny<0 || ny>M-1) continue;

                if(!visited[nx][ny] && graph[nx][ny]==1) {
                    q.offer(new int[]{nx,ny});
                    cnt[nx][ny] = cnt[x][y]+1;
                    visited[nx][ny] = true;
                }
            }
        }

        return cnt[N-1][M-1];
    }
    
}
```

- cnt[][] 아이디어를 떠올려야 한다
- 실수노트 : 문제조건에 첫번째칸도 포함한다는걸 놓쳤다

```
import java.io.*;
import java.util.*;


public class Main{

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken()); // 노드
        int M = Integer.parseInt(st.nextToken()); // 엣지
        int K = Integer.parseInt(st.nextToken()); // 타겟하는 최단거리
        int X = Integer.parseInt(st.nextToken()); // 출발노드 
        

        List<Integer>[] adj = new List[N+1];
        for(int i=1;i<=N;i++) {
            adj[i] = new ArrayList<>();
        }

        for(int i=0;i<M;i++) {
            st = new StringTokenizer(br.readLine());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            adj[a].add(b);
        }


        Queue<Integer> q = new ArrayDeque<>();
        int[] cnt = new int[N+1]; // 해당 노드는 몇번째에 도달 
        q.offer(X);
        Arrays.fill(cnt,-1);
        cnt[X]=0;
        
        while(!q.isEmpty()) {
            int u = q.poll();


            for(int v:adj[u]) {
                if(cnt[v]==-1) { // 실수노트 : 0을 미방문으로 하면 출발지를 다시 방문해서 엣지케이스 생김. 그렇다고 출발지만 -1로 하면 방문횟수가 달라짐. 그래서 전체를 -1로 하고 X를 0으로 초기화
                    cnt[v] = cnt[u] + 1;
                    q.offer(v);
                }
            }
        }

        List<Integer> ans = new ArrayList<>(); 

        for(int i=1;i<=N;i++) {
            if(cnt[i]==K) ans.add(i);
        }

        if(ans.isEmpty()) bw.write("-1");
        else {

            for(int a:ans) {
                bw.write(a+"\n");
            }
        }
        bw.flush();
        bw.close();


    }



}
```

- 실수노트 : 0을 미방문으로 하면 출발지를 다시 방문해서 엣지케이스 생김. 그렇다고 출발지만 -1로 하면 방문횟수가 달라짐. 그래서 전체를 -1로 하고 X를 0으로 초기화
- 오름차순으로 for문을 돌기때문에 정렬은 필요없음 (시간복잡도 단축)

```
벽3개 만들고, 바이러스 퍼트리고, 0 갯수 최댓값 갱신 

*/
public class Main{
    static List<int[]> empty;
    static int ans = -1;
    static int[] combi;
    static List<int[]> virus;
    static int[][] graph;
    static int N;
    static int M;
    static int[] dr = {-1,1,0,0}; // 상하좌우
    static int[] dc = {0,0,-1,1};


    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken()); // 세로
        M = Integer.parseInt(st.nextToken());// 가로

        virus = new ArrayList<>();
        empty = new ArrayList<>();
        combi = new int[3];

        graph = new int[N][M];
        for(int i=0;i<N;i++) {
            st = new StringTokenizer(br.readLine());
            for(int j=0;j<M;j++) {
                graph[i][j] = Integer.parseInt(st.nextToken());
                if(graph[i][j]==0) empty.add(new int[]{i,j});
                if(graph[i][j]==2) virus.add(new int[]{i,j});
            }
        }

        wallCombination(0,0);

        bw.write(ans+"");
        bw.flush();
        bw.close();


    }

    // 벽조합마다 getNumEmpty호출
    static void wallCombination(int depth, int start) { // depth개 골랏음. start부터 시작.
        if(depth==3) {
            int num = getNumEmpty();
            ans = Math.max(ans, num);
            return ;
        }

        for(int i=start;i<empty.size();i++) {
            combi[depth] = i;
            wallCombination(depth+1,i+1);
        }
    }

    // 벽3개 세우고, 바이러스 퍼트리고, 0갯수 반환
    static int getNumEmpty() {
        int[][] copy = new int[N][M];
        int cnt = 0;

        for(int i=0;i<N;i++) {
            for(int j=0;j<M;j++) {
                copy[i][j] = graph[i][j];
            }
        }
        for(int i=0;i<3;i++) {
            int idx = combi[i];
            int[] wall = empty.get(idx);
            int r = wall[0];
            int c = wall[1];
            copy[r][c] = 1;
        }

        Queue<int[]> q = new ArrayDeque<>();
        for(int[] viru:virus) {
            q.offer(viru);
        }

        while(!q.isEmpty()) {
            int[] curr = q.poll();
            int r = curr[0];
            int c = curr[1];

            for(int i=0;i<4;i++) {
                int nr = r + dr[i];
                int nc = c + dc[i];

                if(nc<0 || nc>=M || nr<0 || nr>=N) continue; // 장외

                if(copy[nr][nc]==0) {
                    copy[nr][nc] = 2; // 감염
                    q.offer(new int[]{nr,nc});
                }
            }
        }

        for(int i=0;i<N;i++) {
            for(int j=0;j<M;j++) {
                if(copy[i][j]==0) cnt++;
            }
        }

        return cnt;

    }


}
```

```
낮은번호 바이러스부터 증식

        int[][] scd = new int[N+1][N+1];
        for(int i=1;i<=N;i++) {
            st = new StringTokenizer(br.readLine());
            for(int j=1;j<=N;j++) {
                map[i][j] = Integer.parseInt(st.nextToken());
                if(map[i][j]>0) {
                    viruss.add(new int[]{map[i][j],i,j});
                }
            }
        }

        Collections.sort(viruss,(a,b)-> Integer.compare(a[0],b[0]));

        st = new StringTokenizer(br.readLine());

        int X = Integer.parseInt(st.nextToken());

        for(int i=0;i<viruss.size();i++) {
            q.offer(viruss.get(i));
        }

        while(!q.isEmpty()) {
            int[] curr = q.poll();
            int virusNum = curr[0];
            int r = curr[1];
            int c = curr[2];

                    scd[nr][nc] = scd[r][c]+1;
```

```
import java.util.*; 

/*
올바른 괄호 문자열 검사 메서드 

1~4번 실행 메서드
*/
class Solution {
    public String solution(String p) {
        String answer = "";     
        answer = fix(p);
        
        return answer;
    }
    
    // 1~4번 수행
    static String fix(String str) {
        if(str.length()==0) return "";
        int a=0;
        int b=0;
        String u="";
        String v="";
        StringBuilder sb = new StringBuilder();
        
        for(int i=0;i<str.length();i++) {
              
            if(a==b && a!=0) { // u분리 완료
                if(str.charAt(i)=='(') {
                    sb.append("(");
                }
                else {
                    sb.append(")");
                }
            }
            else {
                if(str.charAt(i)=='(') {
                    a++;
                    sb.append("(");
                }
                else {
                    b++;
                    sb.append(")");
                }
                if(a==b && a!=0) {
                    u = sb.toString();
                    sb.setLength(0);
                }
            }
        
        }
        v = sb.toString();
        sb.setLength(0);
        
        // 3번
        if(isRight(u)) {
            String newV = fix(v);
            sb.append(u);
            sb.append(newV);
            return sb.toString();
        }
        
        
        //4번
        sb.append("(");
        String newV = fix(v);
        sb.append(newV);
        sb.append(")");
        for(int i=1;i<u.length()-1;i++) {
            if(u.charAt(i)=='(') sb.append(")");
            else sb.append("(");
        }
        return sb.toString();
        
    }
    
    // 올바른 괄호 문자열 검사 메서드 
    static boolean isRight(String str) {
        Deque<Character> stack = new ArrayDeque<>();
        
        for(int i=0;i<str.length();i++) {
            if(str.charAt(i)=='(') stack.push('(');
            else {
                if(!stack.isEmpty() && stack.peek()=='(') stack.pop();
                else return false;
            }
        }
        
        return true;
    }
}
```

```
/*

각 조합마다 선생의 감시를 피할 수 있는지 검사한다
하나라도 YES라면 종료
*/
public class Main{
    static List<int[]> wall;
    static int[] combi = new int[3];
    static String ans = "";
    static char[][] map;
    static int N;
    static List<int[]> Teacher;
    static int[] dr = {-1,1,0,0};
    static int[] dc = {0,0,-1,1};

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        N = Integer.parseInt(br.readLine());
        map = new char[N][N];
        wall = new ArrayList<>();
        Teacher = new ArrayList<>();
        for(int i=0;i<N;i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            for(int j=0;j<N;j++) {
                map[i][j] = st.nextToken().charAt(0);
                if(map[i][j]=='X') wall.add(new int[]{i,j});
                else if(map[i][j]=='T') Teacher.add(new int[]{i,j});
            }
        }
        wallCombination(0,0);

        bw.write(ans+"");
        bw.flush();
        bw.close();


    }

    static void wallCombination(int depth, int start) { // 현재 depth개 뽑음, start부터 뽑기 가능
        if(ans.equals("YES")) return;
        
        if(depth==3) {
            ans = getAns();

            return;
        }

        

        for(int i=start;i<wall.size();i++) {
            combi[depth] = i;
            wallCombination(depth+1,i+1);
        }
    }

    // combi에 있는 인덱스에 벽 3개 만들고 선생 감시망 확인
    static String getAns() {
        char[][] copy = new char[N][N];

        for(int i=0;i<N;i++) {
            for(int j=0;j<N;j++) {
                copy[i][j] = map[i][j];
            }
        }

        for(int i=0;i<3;i++) {
            int idx = combi[i];

            int[] coor = wall.get(idx);
            int r = coor[0];
            int c = coor[1];

            copy[r][c] = 'O';
        }

        for(int i=0;i<Teacher.size();i++) {
            int[] tc = Teacher.get(i);
            int r = tc[0];
            int c = tc[1];

            for(int j=0;j<4;j++) {
                int cr = r;
                int cc = c;

                while(true) {
                    cr += dr[j];
                    cc += dc[j];

                    if(cr<0 || cr>=N || cc<0 || cc>=N || copy[cr][cc] == 'O') break;

                    if(map[cr][cc]=='S') return "NO";
                }
            }
        }

        return "YES";

    }

}
```
