# 이미 배운 내용을 훑어보자
- BFS/DFS, 최단경로에서 배운 것들은 그래프 알고리즘의 한 유형
- 나머지 그래프 알고리즘을 여기서 다룬다.
- 그전에 전에꺼 훑어보자 
## 그래프(Graph)와 트리(Tree)의 비교
- 그래프는 노드(Node)와 노드 사이를 잇는 간선(Edge)으로 이루어진 자료구조
- 트리는 그래프의 한 종류이지만 명확한 차이점이 있다

| 구분 | 그래프 (Graph) | 트리 (Tree) |
| :--- | :--- | :--- |
| **방향성** | 방향(Directed) 또는 무방향(Undirected) | 방향 그래프 (컴퓨터공학 기준) |
| **순환성** | 순환(Cycle) 및 비순환(Acyclic) 모두 가능 | **비순환(Acyclic)**만 가능 |
| **루트 노드** | 없음 (모든 노드가 대등함) | **존재함** (하나의 뿌리 노드) |
| **관계** | 부모-자식 관계 없음 | **부모-자식 관계 존재** (계층적) |
| **모델 종류** | 네트워크 모델 (Network Model) | 계층 모델 (Hierarchical Model) |

## 그래프를 구현하는 두 가지 방식
- ① 인접 행렬 (Adjacency Matrix)
  - 방식: 2차원 배열을 사용하여 연결 관계를 표현합니다.
  - 장점: 두 노드의 연결 여부나 간선 비용을 $O(1)$ 시간으로 즉시 확인 가능합니다.
  - 단점: 노드가 $V$개일 때 $O(V^2)$의 메모리 공간이 필요하여 메모리 효율이 낮습니다.
  - 활용: 플로이드 워셜 알고리즘 등.
- ② 인접 리스트 (Adjacency List)
  - 방식: 리스트(혹은 연결 리스트)를 사용하여 연결된 정보만 저장합니다.
  - 장점: 간선의 개수 $E$만큼인 $O(E)$의 메모리만 사용하여 공간 효율적입니다.
  - 단점: 특정 노드 간 연결을 확인하는 데 $O(V)$의 시간이 소요될 수 있습니다.
  - 활용: 다익스트라 최단 경로 알고리즘 등.
```
// 인접 행렬
int[][] dist = new int[N+1][N+1];

// 인접 리스트
ArrayList<Node>[] adj = new ArrayList<>();
```
## 팁
- 문제 식별: "여러 개체가 서로 연결되어 있다"는 표현이 나오면 그래프 알고리즘을 떠올려야 합니다.
- 알고리즘 선택 기준:
  - 노드 개수가 적다면 플로이드 워셜(인접 행렬)이 구현하기 편합니다.
  - 노드와 간선 개수가 모두 많다면 다익스트라(인접 리스트 + 우선순위 큐)를 사용하는 것이 유리합니다.
- 연관 개념: 크루스칼 알고리즘은 그리디, 위상 정렬은 큐/스택 자료구조를 활용하여 구현됩니다.
# 서로소 집합 (유니온 파인드)
- 서로소 집합 : 공통원소가 없는 두 집합
- 서로소 집합 자료구조 : 서로소 부분 집합들로 나누어진 원소들의 데이터를 처리하기 위한 자료구조
- 유니온-파인드 자료구조라고도 부른다
- 연산
  - union : 두 원소가 속한 집합 합치기, 보통 더 작은 원소가 부모가 되도록 구현
  - find : 어떤 집합에 속했는지 확인 (루트노드 확인)
```
import java.io.*;
import java.util.*;

public class Main {
    static int[] parent;

    // 초기화: 처음에는 자기 자신이 대표자
    static void makeSet(int n) {
        parent = new int[n + 1];

        for (int i = 1; i <= n; i++) {
            parent[i] = i;
        }
    }

    // find: x가 속한 집합의 대표자 찾기 + 경로 압축
    static int find(int x) {
        if (parent[x] == x) {
            return x;
        }

        // 경로 압축 핵심
        parent[x] = find(parent[x]);

        return parent[x];
    }

    // union: 두 집합 합치기
    static void union(int a, int b) {
        int rootA = find(a);
        int rootB = find(b);

        if (rootA == rootB) {
            return;
        }

        // 한쪽 대표자를 다른 쪽 대표자 밑으로 붙임
        parent[rootB] = rootA;
    }

    // 같은 집합인지 확인
    static boolean isSameSet(int a, int b) {
        return find(a) == find(b);
    }
}
```

## 경로 압축 (Path Compression)
- (P) 기본적인 Find 함수는 트리가 한쪽으로 치우칠 경우(예: $1 \leftarrow 2 \leftarrow 3 \leftarrow 4$) 시간 복잡도가 $O(V)$로 늘어날 수 있습니다.
- (S) 경로 압축 기법
- Find 함수를 호출할 때 방문하는 노드들이 직접 루트 노드를 가리키도록 업데이트
```
static int find(int x) {
    if (parent[x] == x) {
        return x;
    }

    parent[x] = find(parent[x]); //   핵심
    return parent[x];
}
```
```
// 더 짧은버전
static int find(int x) {
    if (parent[x] != x) {
        parent[x] = find(parent[x]);
    }
    return parent[x];
}
```

## 사이클 판별 (Cycle Detection)
- 그래프의 모든 간선에 대해 아래를 수행
  - 간선의 두 노드 Find연산 (루트찾기)
    - 다른 루트면 : 두 노드 Union연산 (합치기)
    - 같은 루트면 : 사이클이 확인된 것
```
import java.io.*;
import java.util.*;

public class Main {
    static int[] parent;

    static int find(int x) {
        if (parent[x] == x) return x;
        return parent[x] = find(parent[x]); // 경로 압축
    }

    static boolean union(int a, int b) {
        int rootA = find(a);
        int rootB = find(b);

        // 이미 같은 집합이면 사이클 발생
        if (rootA == rootB) {
            return true;
        }

        // 합치기
        parent[rootB] = rootA;
        return false;
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        int v = Integer.parseInt(st.nextToken()); // 정점 개수
        int e = Integer.parseInt(st.nextToken()); // 간선 개수

        parent = new int[v + 1];

        for (int i = 1; i <= v; i++) {
            parent[i] = i;
        }

        boolean hasCycle = false;

        for (int i = 0; i < e; i++) {
            st = new StringTokenizer(br.readLine());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            if (union(a, b)) {
                hasCycle = true;
                break;
            }
        }

        if (hasCycle) {
            System.out.println("사이클 발생");
        } else {
            System.out.println("사이클 없음");
        }
    }
}
```

# 신장 트리(Spanning Tree)
> 모든 노드 포함, 사이클 없는, '부분 그래프'
- 정의는 트리랑 비슷한데, 부분 그래프인게 차이점임

## 크루스칼 알고리즘 
> 최소 신장 트리(MST, Minimum Spanning Tree)를 찾는, 그리디(Greedy) 알고리즘
- ex) 도시간에 연결하는 최소 비용
- 핵심: 크루스칼 = 간선을 가중치 오름차순 정렬 → 사이클 안 생기면 선택 → N-1개 선택하면 끝
### 주요 단계
1. **정렬:** 모든 간선 데이터를 비용에 따라 오름차순으로 정렬합니다.
2. **검사 및 합치기:** 정렬된 간선을 하나씩 확인하며, 해당 간선이 노드들 사이에 사이클을 발생하는지 확인합니다.
  - **사이클이 발생하지 않으면:** 최소 신장 트리에 포함시킵니다 (union 연산).
  - **사이클이 발생하면:** 포함시키지 않고 넘어갑니다.
3. **반복:** 모든 간선에 대해 2번 과정을 반복합니다.
```
import java.io.*;
import java.util.*;

public class Main {

    static class Edge implements Comparable<Edge> {
        int from;
        int to;
        int cost;

        Edge(int from, int to, int cost) {
            this.from = from;
            this.to = to;
            this.cost = cost;
        }

        @Override
        public int compareTo(Edge other) {
            return Integer.compare(this.cost, other.cost);
        }
    }

    static int[] parent;

    // 대표 부모 찾기
    static int find(int x) {
        if (parent[x] == x) return x;
        return parent[x] = find(parent[x]); // 경로 압축
    }

    // 두 집합 합치기
    static boolean union(int a, int b) {
        int rootA = find(a);
        int rootB = find(b);

        // 이미 같은 집합이면 사이클 발생
        if (rootA == rootB) return false;

        // 번호 작은 쪽을 부모로
        if (rootA < rootB) {
            parent[rootB] = rootA;
        } else {
            parent[rootA] = rootB;
        }

        return true;
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        int n = Integer.parseInt(st.nextToken()); // 정점 개수
        int m = Integer.parseInt(st.nextToken()); // 간선 개수

        List<Edge> edges = new ArrayList<>();

        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            int cost = Integer.parseInt(st.nextToken());

            edges.add(new Edge(a, b, cost));
        }

        // 1. 간선을 비용 기준 오름차순 정렬
        Collections.sort(edges);

        // 2. 유니온 파인드 초기화
        parent = new int[n + 1];

        for (int i = 1; i <= n; i++) {
            parent[i] = i;
        }

        int result = 0;      // 최소 스패닝 트리 비용
        int edgeCount = 0;   // 선택한 간선 수

        // 3. 비용이 작은 간선부터 확인
        for (Edge edge : edges) {
            // 사이클이 안 생기면 선택
            if (union(edge.from, edge.to)) {
                result += edge.cost;
                edgeCount++;

                // MST는 간선이 n - 1개면 완성
                if (edgeCount == n - 1) break;
            }
        }

        System.out.println(result);
    }
}
```

- 시간복잡도 : O(E log E)
  - 간선 정렬이 핵심
# 위상 정렬
> 사이클이 없는 방향 그래프에서, 노드의 '선후 관계'를 지키며, 노드를 일렬로 나열
ex) 대학교의 선수과목을 고려한 학습 순서
<img width="392" height="113" alt="image" src="https://github.com/user-attachments/assets/7c8b87ec-e7dc-4fc5-b24d-ef9e0018f760" />

- **진입차수(Indegree):** 특정한 노드로 들어오는 간선의 개수를 의미합니다.
## 로직
1. **초기 단계:** 진입차수가 0인 노드를 큐에 넣습니다.
2. **반복 단계 (큐가 빌 때까지):**
  - 큐에서 원소를 꺼내 해당 노드에서 출발하는 간선을 그래프에서 제거합니다.
  - 새롭게 진입차수가 0이 된 노드를 큐에 넣습니다.
3. **결과:** 큐에서 빠져나간 노드를 순서대로 출력하면 위상 정렬 완료입니다

```
import java.io.*;
import java.util.*;

public class Main {

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        int n = Integer.parseInt(st.nextToken()); // 정점 개수
        int m = Integer.parseInt(st.nextToken()); // 간선 개수

        List<Integer>[] graph = new ArrayList[n + 1];
        int[] indegree = new int[n + 1]; // 진입차수

        for (int i = 1; i <= n; i++) {
            graph[i] = new ArrayList<>();
        }

        // 방향 그래프 입력
        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());

            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            // a -> b
            graph[a].add(b);
            indegree[b]++;
        }

        Queue<Integer> queue = new ArrayDeque<>();

        // 진입차수가 0인 노드 먼저 큐에 삽입
        for (int i = 1; i <= n; i++) {
            if (indegree[i] == 0) {
                queue.offer(i);
            }
        }

        List<Integer> result = new ArrayList<>();

        while (!queue.isEmpty()) {
            int now = queue.poll();
            result.add(now);

            // now와 연결된 다음 노드들의 진입차수 감소
            for (int next : graph[now]) {
                indegree[next]--;

                // 새롭게 진입차수가 0이 된 노드 큐에 삽입
                if (indegree[next] == 0) {
                    queue.offer(next);
                }
            }
        }

        // 사이클이 있으면 모든 노드를 정렬할 수 없음
        if (result.size() != n) {
            System.out.println("사이클 발생");
            return;
        }

        StringBuilder sb = new StringBuilder();

        for (int node : result) {
            sb.append(node).append(' ');
        }

        System.out.println(sb);
    }
```

- **사이클 확인:** 모든 원소를 방문하기 전에 큐가 빈다면 사이클이 발생한 것으로 판단

```
import java.io.*;
import java.util.*;


public class Main{

    static int[] parent;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken());
        int M = Integer.parseInt(st.nextToken());

        //초기화 : 처음에는 자기 자신이 부모
        parent = new int[N+1];
        for(int i=0;i<=N;i++) {
            parent[i] = i;
        }

        for(int i=0;i<M;i++) {
            st = new StringTokenizer(br.readLine());
            int oper = Integer.parseInt(st.nextToken());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            if(oper==0) {
                union(a,b);
            }
            else {
                if(find(a)==find(b)) {
                    bw.write("YES\n");
                }
                else {
                    bw.write("NO\n");
                }
            }

        }


        bw.flush();
        bw.close();


    }

    static void union(int a, int b) {
        int rootA = find(a);
        int rootB = find(b);

        if(rootA==rootB) return;

        parent[rootB] = rootA;
    }

    // 노드input 루트output
    static int find(int a) {
        if(parent[a]==a) return a;

        return parent[a] = find(parent[a]);
    }
    

}
```

```
import java.io.*;
import java.util.*;

// 최소신장트리 만든 후, 최고비용 간선 하나 제거하면 두 마을로 분리된다
public class Main{

    static int[] parent;

    static class Edge implements Comparable<Edge> {
        int u;
        int v;
        int w;

        Edge(int u, int v, int w) {
            this.u = u;
            this.v = v;
            this.w = w;
        }

        @Override
        public int compareTo(Edge o) {
            return Integer.compare(this.w, o.w);
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        int N = Integer.parseInt(st.nextToken()); // 노드
        int M = Integer.parseInt(st.nextToken()); // 간선

        //초기화
        parent = new int[N+1];
        for(int i=1;i<=N;i++) {
            parent[i] = i;
        }
        ArrayList<Edge> edges = new ArrayList<>();


        for(int i=0;i<M;i++ ) {
            st = new StringTokenizer(br.readLine());
            int A = Integer.parseInt(st.nextToken()); // 노드
            int B = Integer.parseInt(st.nextToken()); // 노드
            int C = Integer.parseInt(st.nextToken()); // 가중치

            edges.add(new Edge(A,B,C));
        }

        Collections.sort(edges);
        int cost = 0;
        int edgeCnt = 0;

        for(Edge edge:edges) {
            // 사이클이 아니면
            if(union(edge.u,edge.v)) {
                edgeCnt++;
                cost += edge.w;
            }
            if(edgeCnt==N-2) break;
        }


        bw.write(cost+"");
        bw.flush();
        bw.close();


    }


    // 노드input, 같은집합이면false, 다른집합이면 합친후true
    static boolean union(int a, int b) {
        int rootA = find(a);
        int rootB = find(b);

        // 사이클 발생
        if(rootA==rootB) return false;

        parent[rootA] = rootB;

        return true;
    }

    static int find(int a) {
        if(parent[a]==a) return a;

        return parent[a] = find(parent[a]);
    }
    

}
```
