# 범위를 반씩 좁혀가는 탐색
## 순차 탐색(Sequential Search)
> 리스트 안에 특정한 데이터를 찾기 위해 앞에서부터 **순차적**으로 확인하는 방법
- 이진 탐색 전에 순차 탐색을 이해해야 한다
- 보통 정렬되지 않은 리스트에서 사용한다
- ex) 리스트에서  특정 값의 원소가 있는지 확인할 때, 리스트에서 특정 값의 원소의 수를 셀 때 count()

## 이진탐색 : 반으로 쪼개면서 탐색하기
> 찾으려는 데이터와 중간점 데이터를 반복적으로 비교
- 사용하는 변수 (3개)
  - 시작점(Start): 탐색 범위의 맨 앞 인덱스
  - 끝점(End): 탐색 범위의 맨 뒤 인덱스
  - 중간점(Mid): 시작점과 끝점의 중간 인덱스 ($$(시작점 + 끝점) / 2$$, 소수점 이하 버림)
- 정렬된 데이터에만 사용
