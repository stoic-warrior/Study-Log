# 선언형 프로그래밍 (Declarative Programming)
> “무엇을 할지”를 말하고, “어떻게”는 시스템에 맡긴다
- 결과에 집중
- 내부 동작을 직접 제어하지 않음
## ex
- SQL: `SELECT * FROM user WHERE age > 20`
- **무엇을 할지 (결과)**
  - “user 테이블에서”
  - “나이가 20보다 큰 것만”
  - “가져와라”
- **어떻게 할지 (방법)** 
  - 몇 번째 행부터 읽을지
  - 인덱스를 쓸지
  - 정렬을 먼저 할지
  - 반복문을 돌지 말지
  - 👉 DB가 알아서 결정함
