### 21일
- FDX 백엔드 코드 복제
  - ㅡ> git으로 fdx를 서버에 clone함
- 운영 비밀값 생성
  - .env파일 에 다음 값 무작위 생성 (docker compose가 .env읽어서 compose.yaml안의 ${}내용을 채움) 
    - DB_PASSWORD (스프링부트 백엔드가 MySQL에 접속할 때 사용)
    - MYSQL_ROOT_PASSWORD (MySQL의 root계정 비밀번호, 최고 권한 계정) 
    - JWT_SECRET (로그인 후 발급하는 JWT 토큰에 서명하는 비밀키)
  - 권한 제한
    - chmod 600 .env (파일 소유자만 읽기, 쓰기 가능) 
  - git에 올라가지 않도록 설정
    - echo ".env" >> .git/info/exclude
