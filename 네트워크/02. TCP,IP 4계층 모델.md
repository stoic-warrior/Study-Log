# 개요
- **인터넷 프로토콜 스위트(Internet Protocol Suite)** = 인터넷에서 컴퓨터들이 서로 정보를 주고받을 때 사용하는 **프로토콜의 집합**이다.
- 이를 설명하는 모델이 두 가지가 있다:
  - **TCP/IP 4계층 모델**
  - **OSI 7계층 모델**
- 여기서는 **TCP/IP 모델**을 중심으로 설명한다.
