# 운영체제의 역할과 구조
> 하드웨어를 관리하고 사용자에겐 편의를 제공하는 시스템
## 운영체제의 역할
- **CPU/프로세스 관리**: CPU 스케줄링, 프로세스 생성·삭제, 자원 할당/반환
- **메모리 관리**: 프로세스별 메모리 할당·회수
- **디스크/파일 관리**: 파일 저장 방식, 파일 시스템 관리
- **I/O 디바이스 관리**: 키보드·마우스·디스크 같은 입출력 장치와 프로그램 사이의 데이터 전송을 관리
