# 🧠 프로세스(Process) — "운영체제의 핵심 전투 단위"

---

# ✅ 프로세스란 무엇인가?
- 실행 중인 프로그램(Program in Execution)
- 운영체제(OS)에서 할당된 자원들과 함께 관리되는 실행 단위
- 프로세스 = 코드(명령어) + 데이터 + 자원(메모리, 파일 등) + 제어 흐름(CPU 상태)
